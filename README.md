# center.news

The IOS application is based on the use News API. (https://newsapi.org)

Stack:
- SwiftUI
- Combine
- Core Data
- MVVM

Features:
- news list, opens in safari
- search news from the list
- add news to bookmarks
- removing news from bookmarks
- removing using swipe actions
